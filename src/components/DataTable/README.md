# DataTable

Generic, reusable TanStack Table wrapper (shadcn + Tailwind) with:

- Search (globalFilter)
- Sort, Filter, Pagination + pageSize selector
- Row actions (dropdown)
- Expand rows (sub component) + support for nested sub rows
- Server mode hooks for API integration (query change, infinite scroll)

Usage: import `DataTable` from `src/components/DataTable`.
